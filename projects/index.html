    • Top Down Shooter - Zombie Survival

      A zombie survival top down shooter game i created to understand the 3D concepts of Unity3D. * Usage of Mechanim system to setup the animation transitions for user inputs. * A state machine for NPC. StateBehaviour scripts for zombie NPC were implement for actions such as wander, chase and attack. * NavMesh to create traversal space for the NPCS. * Setup NavMeshAgent for NPC and access it via script to handle its movement based on its state. * Usage of Animation Events to execute required code. * Particle System for bullet tracer, blood spill.
